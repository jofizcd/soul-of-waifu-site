# Генерация изображений

> Руководство по настройке визуализации: интеграция с локальными (Automatic1111, ComfyUI, FLUX) и облачными (NovelAI, DALL-E 3) генераторами изображений.

В Soul of Waifu встроен продвинутый модуль создания иллюстраций. Ваши персонажи могут не просто общаться с вами голосом и текстом, но и присылать по вашей просьбе свои изображения, наглядно иллюстрируя происходящие в ролевой игре события. 

Все сгенерированные изображения автоматически сохраняются в индивидуальную галерею персонажа на вашем диске.

---

## Поддерживаемые провайдеры генерации

Система поддерживает пять популярнейших движков генерации на любой вкус — от полностью локального запуска до облачных нейросетей коммерческого уровня:

| Провайдер | Тип | Стиль | Что требуется для запуска |
|---|---|---|---|
| **Automatic1111** | Локальный | Универсальный | Запущенная на ПК сборка WebUI с включенным флагом API эндпоинта. |
| **ComfyUI** | Локальный | Универсальный | Запущенная сборка ComfyUI, настроенная на совместимость с API Automatic1111. |
| **FLUX** | Облачный | Фотореализм | Сверхдетализированная модель. Требует наличия ключа FAL API Token. |
| **NovelAI** | Облачный | Аниме-арт | Лучший в мире генератор аниме-графики. Требует платного API-ключа. |
| **DALL-E 3** | Облачный | Универсальный | Генератор от OpenAI. Требует стандартного ключа OpenAI API. |

## Как устроена генерация? Формула промпта

Когда вы просите персонажа прислать картинку, программа автоматически собирает итоговый запрос (промпт) из трех частей:

```
    [Префикс качества (Prefix)] + [Основной запрос (Core Prompt)] 
                                ↓
                 [Итоговый промпт отправляется ИИ]
                                ↓
               [Применяется Негативный промпт (Negative)]
```

*   **Префикс качества (Prefix Prompt):** Задается в глобальных настройках. Это набор дежурных тегов для улучшения картинки (например: *«masterpiece, best quality, highly detailed, vivid colors»*). Он автоматически подмешивается в начало любого запроса.
*   **Основной запрос (Core Prompt):** Формируется языковой моделью на основе того, что происходит в чате прямо сейчас (например, описание позы персонажа, его одежды или локации).
*   **Негативный промпт (Negative Prompt):** Список вещей, которых на картинке быть не должно (например: *«ugly, deformed, bad anatomy, blurry»*). Используется для локальных моделей и NovelAI.

## Шаг за шагом: Настройка локального генератора (Automatic1111 / ComfyUI)

Если вы хотите генерировать картинки полностью бесплатно на своей видеокарте:

1.  Установите и запустите локальный веб-интерфейс **Automatic1111** (Stable Diffusion) или **ComfyUI**.
2.  При запуске обязательно укажите флаг `--api`, чтобы Soul of Waifu могла общаться с сервером генератора.
3.  В настройках Soul of Waifu перейдите во вкладку **Image Generation**.
4.  Укажите адрес вашего локального сервера (по умолчанию для Automatic1111 это `http://127.0.0.1:7860`).
5.  Выберите желаемое разрешение картинок (например, 512x768 или 1024x1024 для моделей FLUX) и настройте количество шагов генерации.

::: tip
**Оптимизация под видеокарты:**
Если у вас мало видеопамяти (VRAM), одновременный запуск текстовой модели и генератора изображений (Stable Diffusion / FLUX) может вызвать вылет программы. В таком случае рекомендуется использовать облачные провайдеры (DALL-E 3 или Novel AI) или выгружать больше слоев текстовой модели на процессор (CPU).
:::

## Куда сохраняются изображения?

Все созданные в процессе общения изображения автоматически складываются в личную папку вашего чата:
`app/data/.soul/Имя_Персонажа/chats/ID_чата/images/`

Вы можете в любой момент открыть галерею чата, чтобы пересмотреть все присланные арты, или заглянуть в папку на диске через стандартный проводник Windows.
